Roles Tagging Feature
 
Product Description
A new platform-wide Roles tagging system that enhances how we manage, organize, and search for roles. Role Tagging System 2.0 must be able to achieve 3 key objectives:
  • Make tags more scalable
  • Give tags actual meaning
  • Is effective for organizing departments/teams/units of roles
